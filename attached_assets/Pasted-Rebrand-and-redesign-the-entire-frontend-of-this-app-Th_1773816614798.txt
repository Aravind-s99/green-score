STEP 1 — LANDING PAGE (LandingPage.jsx)
==================================================
Create src/pages/LandingPage.jsx — a full-screen hero 
that shows BEFORE the app. User scrolls down to enter.

Structure:
- Fixed background: dark forest (#0a1f0e) with a subtle 
  noise texture overlay (use CSS background-image with 
  a repeating SVG noise pattern at 3% opacity)
- Centered content:
  * Logo: a simple leaf SVG icon in gold above the title
  * Title: "CanopyScore" in a serif font (use Google Font 
    "Playfair Display") — large, 72px, gold color (#c9a84c)
  * Tagline below: "AI-powered truth for carbon markets" 
    in 18px, --text-muted color, letter-spacing: 0.15em
  * A thin gold horizontal rule (width: 120px, margin: auto)
  * Short description (2 lines max): 
    "Every green project. Verified by satellite data, 
    public registries, and transparent AI scoring."
  * Scroll indicator at bottom: animated gold chevron 
    pointing down with text "scroll to explore" 
    (use CSS keyframes to make it bounce gently)

- On scroll past 100vh, smoothly transition to the main app 
  using IntersectionObserver. The transition should feel like 
  parting through a forest canopy — use a CSS clip-path 
  animation: clip-path goes from circle(0%) to circle(150%) 
  over 800ms ease-in-out when triggered.

STEP 2 — FALLING LEAVES + LOADING ANIMATION
==================================================
Create src/components/LeafLoader.jsx

Falling leaves animation:
- Generate 12 leaf shapes using SVG paths 
  (simple 3-point leaf silhouettes, not emoji)
- Each leaf: random size (16-32px), gold or green color, 
  starts at random x position at top, falls to bottom 
  with a gentle swaying CSS animation
- Use @keyframes with transform: translateY + rotate + 
  translateX(sin wave) to make them sway as they fall
- Stagger animation-delay randomly per leaf (0 to 3s)
- Loop infinitely

Tree growing animation:
- In the center, draw an SVG tree that "grows" on load
- Start: just roots/stump visible
- Use stroke-dashoffset animation on SVG paths to make 
  the trunk and branches draw themselves over 2 seconds
- Then fade in a circle of leaves (green filled circles) 
  around the crown after trunk finishes drawing
- Below the tree: "Analysing project..." text in gold, 
  with the words cycling through:
  "Fetching registry data..."
  "Checking satellite imagery..."  
  "Running AI scoring..."
  "Building evidence trail..."
  (change every 1.5s using setInterval)

Show LeafLoader whenever a project score is loading.
